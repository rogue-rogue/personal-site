CSS stands for Cascading Style Sheet 

The style sheet holds the rule set

Rule sets contain one or more of the following; 

selector {declaration} => selector {property: value; property:value ...}

The rule set can live in one of 3 places: 
1. Inline via elementâ€™s opening tag style attribute:
    <p style="color: blue; font-size: 18px;">Blue, 18-point text</p>

2. Header via style tag:
    <head>
      <style>
        h1 {
          color: blue;
        }
      </style>
    </head>

3. Separate file via <link> tag in the <head>:
    <head>
      <link href="style.css" type="text/css" rel="stylesheet">
    </head>
    Note: href = url type = content type, rel = relationship to html file

Option 3 is easiest to run at scale.
