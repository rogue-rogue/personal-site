Learning Summary

HTML kicks off with a Document Type Declaration. This is is the instruction to the browser about the type of document and the version of HTML
After this declaration we introduce the first element: HTML Element
The root of an HTML document, all content/structure for an HTML document should be contained between the opening and closing tags.
We then start to fill out parts of the structure. It's got a head and a body.

Head

The head contains all the metadata which doesn't actually appear on the page. This could include a title, html native style specifiers and a link to a css style sheet.

Body

This is where we store the elements that should actually be displayed.
The most foundational of these elements are basic text styling:

  • Headings h1, h2, h3, h4, h5
  • Paragraphs/Blocks of text p
  • Italics/emphasis em
  • Strong/bold strong

We can add line breaks with br --> opener only tags
We can also put these elements into containers *cue the ooohs*. These containers are exactly as they sound. There are two containers I'm aware of currently.
  1. div elements can contain flow content such as headings, paragraphs, links, images, etc.
  2. span elements separate pieces of text within paragraphs so they can be differently formatted

We can implement lists of items such as ordered lists via ol and unordered lists via ul. We can implement multimedia content such as videos and images.
